## Preview

Incognito mode doesn't stop fingerprinting. Your browser leaks:
- Canvas fingerprint
- WebGL renderer
- Font list
- Screen resolution + timezone combo
- Audio context hash

Even without cookies, you're uniquely identifiable in ~94% of cases.
